## Shared adversarial fixture set

`adversarial/` is the shared sanitizer contract fixture set required by
`_planning/architecture.md` ("SVG sanitization"): script injection
(`script-injection.svg`), event handlers (`event-handlers.svg`), and external
references (`external-references.svg`). Each file also carries stable
`{region}__{component}_{idx}` ids and legitimate internal references
(`#local-symbol`, `url(#leak-pattern)`) that MUST survive sanitization
byte-identical.
